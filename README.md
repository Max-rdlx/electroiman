### Electroimán — proyecto web

Descripción

Esta carpeta contiene una página educativa sobre electroimanes (en español). Incluye una demo interactiva simple para visualizar cómo cambia la "fuerza" al variar la corriente simulada.
tambien cuenta con el archivo soy.html que dice quien soy.

### Archivos clave

- html/index.html — página principal (ábrela en el navegador).
- css/style.css — estilos de la página.
- html/demo.js — script para la demo interactiva.


### Cómo probarlo

1. Abre el archivo `html/index.html` en tu navegador (doble clic o "Abrir con" -> navegador).
2. Usa el control deslizante en la sección "Demo interactiva" para ver los efectos visuales.
